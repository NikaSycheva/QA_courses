2.  Создать папку = mkdir folder
3.  Зайти в папку = cd folder
4.  Создать 3 папки = mkdir folder1 folder2 folder3
5.  Зайти в любую папку = cd pesny_buzovoy
6.  Создать 5 файлов (3txt, 2json) = touch file1.txt file2.txt file3.txt file4.json file5.json
7.  Создать 3 папки = mkdir papka1 papka2 papkatebenemamka
8.  Вывести список сождержимого папки = ls
    Вывести содержимое папки вместе со скрытыми файлами = ls -la
9.  Открыть любой txt файл = vim file.txt
10. Написать что-нибудь в файл = i (insert) и ввод текста
11. Сохранить и выйти = esc :wq 
12. Выйти из папки на уровень выше = cd ..
13. Переместить любые 2 файла, которые вы создали, в любую другую папку = mv nika.png file.txt полный адрес папки, в котрую копируем
14. Скопировать любые 2 файла, которые вы создали, в любую другую папку = cp file1.txt file2.json полный адрес папки
15. Найти файл по имени = find -name 'file.txt'
16. Посмотреть содержимое в реальном времени
tail -f file.txt
cat 2.txt
cat 2.txt | grep
17. Вывести несколько первых строк из текстового файла = head -n3 2.txt
18. Вывести несколько последних строк из текстового файла = tail -n4 2.txt
tail имя_файла = выведет последние 10 строк файла
tail путь к файлу = аналогично выведет последние 10 строк файла
19. Посмотреть содержимое длинного файла = less file.log
less +5 file.log = вывести последние 5 строк файла 
less +/text file.log = поиск указанного текста в файле
